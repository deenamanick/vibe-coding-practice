### Prerequisites

- Complete Module 5 (Node.js & Express)
- Basic understanding of APIs
- A free account on [Cloudinary](https://cloudinary.com)

### 🆘 Why Cloudinary?

**Cloudinary is perfect for students:**
- ✅ **Completely FREE** generous tier
- ✅ **No credit card** required to sign up
- ✅ **Automatic optimization** (makes images fast)
- ✅ **Simple API** for Node.js
- ✅ **Built-in security** features

### Tech Stack

- **Backend**: Node.js, Express, Multer, Cloudinary SDK
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (or Lovable AI)
- **Cloud Storage**: Cloudinary (Free Tier)
- **UI/UX**: Modern, responsive design with image previews
