### 🚀 Why CI/CD?

**Automation is a developer's best friend:**
- ✅ **No Manual Errors**: The computer runs tests and deploys exactly the same way every time.
- ✅ **Faster Feedback**: Find out if your code is broken 1 minute after pushing it.
- ✅ **Ship Faster**: Automatically update your live website the moment you finish a feature.
- ✅ **Free for Students**: GitHub Actions has a massive free tier for public repositories.

### Tech Stack

- **Automation**: GitHub Actions
- **Scripting**: YAML
- **Testing**: Jest / Python Unittest
- **Deployment**: Vercel / Railway / Render
- **UI/UX**: Pipeline status indicators and build logs
