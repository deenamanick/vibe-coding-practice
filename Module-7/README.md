# Module 7: Database Integration & AI Chat

This module teaches you how to connect your backend to a database and build AI chat features.

## Practicals

- `module-7-1-database-setup.md` — Database setup and basic CRUD operations
- `module-7-2-ai-chat-api.md` — Building AI chat API endpoints with Groq
- `module-7-3-chat-history.md` — Storing and retrieving chat history
- `module-7-4-user-chat-sessions.md` — Managing user-specific chat sessions
- `module-7-5-ai-chat-ui.md` — Frontend for AI chat with history

### Recommended order

1. Database setup
2. AI chat API
3. Chat history
4. User chat sessions
5. AI chat UI

### Prerequisites

- Complete Module 6 (Authentication & Authorization)
- Basic understanding of REST APIs
- Node.js and Express.js experience

### Tech Stack

- **Backend**: Node.js, Express, SQLite3, Groq AI
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Database**: SQLite with proper schema design
- **Authentication**: JWT-based user management
- **AI Integration**: Groq (free, fast models)
- **UI/UX**: Modern, responsive design with animations
