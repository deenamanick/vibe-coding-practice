## Module 5: REST API + CRUD (Node.js + Express)

This module is practice-first. You’ll build and test a tiny REST API using **Node.js + Express**.

### Why this module

Most apps become useful when the UI can read/write data. REST APIs are the simplest way to do that.

### What you will practice

- REST API basics (resources, routes, status codes)
- CRUD operations (Create/Read/Update/Delete)
- Express fundamentals (routes, request/response)
- Middleware (logging, JSON parsing, validation)
- Error handling (consistent error responses)
- Environment variables (ports, modes)
