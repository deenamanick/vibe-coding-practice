# Module 1: Foundational Web Development & Database Basics

Welcome to Module 1! This module introduces you to the core building blocks of modern web applications: **Frontend (HTML/CSS/JS)**, **Backend (API Requests)**, and **Databases (SQL/Docker)**.

## 📂 Folder Contents

- **`module1-1-Frontend.md`**: Project - "Student Introduction Page". Learn HTML structure, CSS styling (Flexbox, Gradients), and basic JavaScript interactivity.
- **`Module-1-Backend.md`**: Project - "Simple Joke Generator". Introduces `fetch()`, external APIs, JSON data, and asynchronous JavaScript (`.then()`).
- **`Module-1-DB.md`**: Project - "Simple Student Database". Learn to run MySQL via Docker and perform fundamental SQL operations (CRUD).
- **`Module-1-NoSQL.md`**: Introduction to non-relational databases.
- **`module-1-Api.md`**: Concepts of Application Programming Interfaces.
- **`module-1-protocol.md` & `module-1-request-response.md`**: Understanding how the internet works (HTTP, Request/Response cycle).
- **`module-1-authentication.md` & `module-1-deployment.md`**: Basic concepts of user security and putting your app online.

## 🚀 Key Learning Paths

### 1. Frontend & Logic
Build interactive user interfaces.
*   **Concepts**: HTML Tags, CSS Glassmorphism, JS Functions, Event Listeners.
*   **Project**: A greeting page that responds to user input.

### 2. Networking & APIs
Connect your app to the world.
*   **Concepts**: `fetch()`, REST APIs, JSON format, Promises.
*   **Project**: A generator that pulls real-time data from an external server.

### 3. Data Persistence (SQL & Docker)
Save and manage information.
*   **Concepts**: Docker Containers, SQL Queries (SELECT, INSERT, UPDATE, DELETE), Primary Keys.
*   **Project**: A student management system running in a virtualized MySQL environment.

## 🛠 Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+), SQL.
- **Tools**: VS Code, Docker, Chrome DevTools.
- **APIs**: Official Joke API.
