## Module 6: Authentication & Authorization (Practice)

This module is about letting users:

- create an account (signup)
- login
- login with a social provider (OAuth)
- understand headers (Basic, Bearer) and SSO
- call protected APIs
- access different features based on role (admin vs user)

### Practicals

- `module-6-1-signup-login.md` — Signup & Login flow
- `module-6-2-oauth-basics.md` — OAuth 2.0 basics (GitHub login flow)
- `module-6-3-auth-schemes.md` — Basic vs Bearer vs JWT vs OAuth2 vs SSO
- `module-6-4-protecting-routes.md` — Protecting routes (auth middleware)
- `module-6-5-role-based-access.md` — Role-based access control (RBAC)

### Recommended order

1. Signup/Login
2. OAuth basics
3. Auth schemes (Basic/Bearer/JWT/SSO)
4. Protect routes
5. Role-based access
